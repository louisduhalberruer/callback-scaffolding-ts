# callback Scaffolding TS

**Install**

Install [Node.js](https://nodejs.org/) and [pnpm](https://pnpm.io/)

```sh
cd ./callback-scaffolding-ts
```

```sh
pnpm install
```

```sh
pnpm dev
```

<br>

**Format**
```sh
pnpm format
```

**Build**
```sh
pnpm build
```

**Preview build**
```sh
pnpm preview
```
